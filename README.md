# Vue 3 + TypeScript + Vite + MasterCSS Chrome Extension Template

## Usage
```
pnpx degit 0miles/vite-vue3-mastercss-chrome-extension my-project
cd my-project
pnpm i
```

## Development
```
pnpm dev
```

## Build
```
pnpm build
```
